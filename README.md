# E-Ticaret Sitesi

ASP.NET Core MVC tabanlı modern e-ticaret platformu.

## Özellikler

- Ürün katalogu ve kategori sistemi
- Kullanıcı kimlik doğrulama (ASP.NET Identity)
- Sepet ve alışveriş sistemi
- Ürün arama ve filtreleme
- Responsive tasarım
- Türk Lirası desteği

## Kullanılan Teknolojiler

- ASP.NET Core 9.0
- Entity Framework Core 9.0
- SQLite
- Bootstrap 5
- jQuery
- Font Awesome

## Kurulum

1. Projeyi klonlayın
2. .NET 9.0 SDK'yı yükleyin
3. Veritabanını oluşturun:
   ```bash
   dotnet-ef database update
   ```
4. Projeyi çalıştırın:
   ```bash
   dotnet run
   ```
